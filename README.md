# VizA11y - Chart Accessibility Transformer

## Overview
A browser extension that automatically detects and transforms data visualizations to be accessible for colorblind users.

## Features
- Automatic chart detection on webpages
- Pattern/texture overlay for data series
- Smart labeling with direct value annotations
- Colorblind profile support (8 types)
- Export transformed charts as PNG/PDF

## Technologies
- TypeScript, D3.js for chart manipulation
- TensorFlow.js for chart detection
- Chrome Extension API
- Python FastAPI (optional backend)
